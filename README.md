VulnScanner

A modern, full‑stack web‑based vulnerability scanner built with FastAPI.

Includes:



• Port scanning

• Security header analysis

• TLS/SSL inspection

• XSS / SQLi tests

• Real‑time scan updates (WebSockets)

• User accounts + admin panel

• Email verification

• Password reset

• Analytics dashboard with charts

• Dark mode + mobile‑responsive UI

• API documentation page



This project is designed to be deployed easily on Render.com.



Features

✔ Vulnerability Scanning

Detects open ports

Scans for missing security headers

TLS/SSL certificate + cipher checks

XSS testing

SQL injection testing

Saves results in database

Live scan updates using WebSockets

✔ Authentication

User accounts

Email verification (Mailjet)

Password reset

Login sessions via cookies

✔ Admin Panel
